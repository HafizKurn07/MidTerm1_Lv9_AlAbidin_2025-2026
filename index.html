ICT Examination Midterm 1 Level 9 Al-Abidin

Selamat mengerjakan ujian praktik ICT. Baca semua poin dengan teliti sebelum memulai.

Tujuan Ujian

Ujian praktik ini bertujuan untuk menguji kemampuan teknis dan problem solving siswa dalam merancang dan membuat tampilan website menggunakan HTML dan CSS.

Mekanisme Ujian
  • Siswa mengerjakan soal berupa studi kasus.
  • Siswa menggunakan laptop pribadi dengan code editor (VSCode atau sejenisnya).
  • Siswa boleh mencari referensi internet/buku, tetapi tidak boleh membuka AI (ChatGPT, Copilot, dll).
  • Penilaian berdasarkan kelengkapan, kualitas kode, styling, dan manajemen waktu.
Ketentuan File Ujian
  • File HTML utama (*.html).
  • Asset gambar (jika ada).
  • Struktur folder yang rapi dan terorganisir.
Aturan Penting
  • Dilarang menggunakan AI generatif (ChatGPT, Copilot, dll).
  • Dilarang menggunakan joki (pengerjaan oleh pihak lain).
Instruksi
  1. Baca studi kasus dengan cermat.
  2. Kerjakan tugas sesuai perintah.
  3. Gunakan HTML + CSS.
  4. Simpan file hasil kerja sesuai format nama yang ditentukan.
Alokasi Waktu

90 menit (1 jam 30 menit)

Studi Kasus

Anda diminta membuat sebuah halaman Formulir Pendaftaran Ekstrakurikuler menggunakan HTML dan CSS dasar. Halaman ini bertujuan melatih pemahaman struktur form HTML, penggunaan berbagai input, serta styling sederhana dengan CSS.

Tugas
  • Buat halaman HTML berjudul “Extracurricular Registration”.
  • Formulir harus memuat field berikut:
    • Nama Lengkap (text)
    • Email (email)
    • Tanggal Lahir (date)
    • Jenis Kelamin (radio: Laki-laki / Perempuan)
    • Pilihan Ekstrakurikuler (select: Futsal, Basketball, Pramuka, Klub Ilmiah)
    • Hari Latihan (select: Senin–Minggu)
    • Waktu Latihan (time)
    • Persetujuan (checkbox dengan teks: “Saya setuju dengan syarat dan ketentuan”)
  • Buat tombol submit dengan teks “Daftar”.
  • Gunakan CSS sederhana untuk:
    • Memberi padding, border, dan rounded pada input.
    • Mengatur warna fokus (focus state) pada input.
    • Memberikan shadow dan hover effect pada tombol.
    • Membuat tampilan form responsif dan rapi di layar laptop maupun HP.
Tantangan (Opsional)
  • Tambahkan animasi hover pada tombol Daftar.
  • Tambahkan warna latar belakang lembut agar tampilan lebih menarik.
